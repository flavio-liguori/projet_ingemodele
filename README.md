# Pipeline IDM : Analyse & Correction Ecore via Mistral AI

Ce projet implémente un pipeline d'Ingénierie Dirigée par les Modèles (IDM) automatisé. Il permet d'analyser, de diagnostiquer et de proposer des corrections sur des modèles `.ecore` (EMF) en utilisant une analyse RCA (Root Cause Analysis) assistée par le LLM **Mistral AI**.

2. **Rendre le script exécutable :**
   ```bash
chmod +x run_pipeline.sh```

3. **Mettre la clé api mistral en var d'environnement :**
   ```bash
export MISTRAL_API_KEY="votre_cle_api_ici" ```
